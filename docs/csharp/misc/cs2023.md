# <a name="compiler-warning-level-1-cs2023"></a>编译器警告（等级 1）CS2023

/noconfig 选项是在响应文件中指定的，因此被忽略  
  
 在响应文件中指定了 [-noconfig](../language-reference/compiler-options/miscellaneous.md#noconfig) 编译器选项，这是不允许的。  
  
 [**DisabledWarnings**](../language-reference/compiler-options/errors-warnings.md#disabledwarnings)选项无法禁止显示此警告。
