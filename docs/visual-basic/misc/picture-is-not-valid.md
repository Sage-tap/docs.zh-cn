# <a name="picture-is-not-valid"></a>图片无效

你尝试向窗体或控件的 `Picture` 属性分配的图形格式不是位图、图标，也不是 Windows 图元文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保尝试加载到属性中的文件 `Picture` 是 Visual Basic 支持的有效图形文件。
