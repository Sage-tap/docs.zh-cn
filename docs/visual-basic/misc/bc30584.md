# <a name="type-cannot-be-inherited-more-than-once"></a>" \<type> " 不能被多次继承

重复了 `Inherits` 语句。  
  
 **错误 ID：** BC30584  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余的 `Inherits` 语句。
