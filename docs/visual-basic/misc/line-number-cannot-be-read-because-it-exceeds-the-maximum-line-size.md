# <a name="line-number-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>无法读取行 \<number>，因为它超过了最大行大小。

无法读取行，因为它超过了最大行大小。 通常这意味着在文件中有一个左引号缺少对应的右引号。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在文件中插入右引号。
