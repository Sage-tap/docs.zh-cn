# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>此关键字已与该集合的某个元素相关联

为集合成员指定的键已标识集合中的另一个成员。 键是方法中指定的字符串 `Add` ，用于唯一标识集合中的特定成员。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 对此成员使用不同的密钥。
