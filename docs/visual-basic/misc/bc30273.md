# <a name="specifier-is-not-valid-on-an-interface-property-declaration"></a>" \<specifier> " 在接口属性声明中无效

接口内的 `Property` 语句包含无效的关键字，如 `Implements`。 接口只能定义成员，而不能实现它们。  
  
 **错误 ID：** BC30273  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从声明语句中删除无效的关键字。  
  
- 将接口成员的实现移动到实现该接口的类。
