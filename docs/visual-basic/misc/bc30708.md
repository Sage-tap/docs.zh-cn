# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>循环语句在“即时”窗口中无效

`Loop` 语句不能在 " **即时** " 窗口中使用。  
  
 **错误 ID：** BC30708  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `Loop` 语句。
