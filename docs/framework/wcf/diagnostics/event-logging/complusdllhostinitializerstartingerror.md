# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError

Id：131  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>说明  

 此事件指示在启动 DllHost 初始值设定项时出现错误。 此事件将列出 AppID、异常、进程名称和进程 ID。
