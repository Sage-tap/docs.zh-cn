# <a name="some-subkeys-cannot-be-deleted"></a>无法删除某些子项

尝试删除注册表项，但操作失败，因为某些子项无法删除。 这通常是由于缺少权限引起的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保您有足够的权限删除指定的子项。
