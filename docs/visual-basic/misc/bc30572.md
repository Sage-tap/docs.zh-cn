# <a name="alias-namespacename-is-already-declared"></a>已声明别名“\<namespacename>”

你已尝试使用 `Imports` 导入已声明的命名空间。  
  
 **错误 ID：** BC30572  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查你尝试导入的命名空间。
