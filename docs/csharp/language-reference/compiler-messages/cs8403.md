# <a name="compiler-error-cs8403"></a>编译器错误 CS8403

带有迭代器块的方法“method”必须为“异步”，才能返回“{IAsyncEnumerable\<T>|IAsyncEnumerator\<T>}”

## <a name="to-correct-this-error"></a>更正此错误

请使用 `async` 修饰符标记方法。
