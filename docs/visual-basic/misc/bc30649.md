# <a name="typename-is-an-unsupported-type"></a>" \<typename> " 是不支持的类型

指定的类型不能由 Visual Studio 表示。  
  
 **错误 ID：** BC30649  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用受支持的类型。  
  
- 开发新类型。
