 <span data-ttu-id="dafa7-106">如果在包含本身不固定类的表达式中使用固定大小的缓冲区，将出此错误。</span><span class="sxs-lookup"><span data-stu-id="dafa7-106">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="dafa7-107">运行时可以自由移动非固定类以优化内存访问，但使用固定大小缓冲区时，这可能会导致错误。</span><span class="sxs-lookup"><span data-stu-id="dafa7-107">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="dafa7-108">若要避免此错误，请此在语句上使用 `fixed` 关键字。</span><span class="sxs-lookup"><span data-stu-id="dafa7-108">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dafa7-109">示例</span><span class="sxs-lookup"><span data-stu-id="dafa7-109">Example</span></span>  

 <span data-ttu-id="dafa7-110">下面的示例生成 CS1666。</span><span class="sxs-lookup"><span data-stu-id="dafa7-110">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
