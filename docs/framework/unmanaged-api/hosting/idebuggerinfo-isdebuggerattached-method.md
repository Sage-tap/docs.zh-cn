# <a name="idebuggerinfoisdebuggerattached-method"></a>IDebuggerInfo::IsDebuggerAttached 方法

获取一个值，该值指示托管调试器是否已附加到此进程。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsDebuggerAttached (  
    [out] BOOL *pbAttached  
);  
```  
  
## <a name="parameters"></a>参数  

 `pbAttached`  
 弄一个指向值的指针， `true` 如果托管调试器附加到进程，则该值为; 否则为 `false` 。
