### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="bbaac-117">使用 EdmGen.exe 为 Visual Basic 项目生成 School 模型</span><span class="sxs-lookup"><span data-stu-id="bbaac-117">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

