# <a name="compiler-error-cs1918"></a><span data-ttu-id="28400-103">编译器错误 CS1918</span><span class="sxs-lookup"><span data-stu-id="28400-103">Compiler Error CS1918</span></span>

<span data-ttu-id="28400-104">无法使用对象初始值设定项为类型为“type”的属性“name”的成员赋值，因为它是值类型。</span><span class="sxs-lookup"><span data-stu-id="28400-104">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="28400-105">尝试使用对象初始值设定项初始化某种结构类型的属性，而该结构类型本身是所初始化的类的属性时，会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="28400-105">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="28400-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="28400-106">To correct this error</span></span>  
  
1. <span data-ttu-id="28400-107">如果必须在对象初始值设定项中完全初始化属性的字段，请将结构更改为类类型。</span><span class="sxs-lookup"><span data-stu-id="28400-107">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="28400-108">否则，在使用对象初始值设定项创建对象之后，在单独方法调用中初始化结构成员。</span><span class="sxs-lookup"><span data-stu-id="28400-108">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28400-109">示例</span><span class="sxs-lookup"><span data-stu-id="28400-109">Example</span></span>  

 <span data-ttu-id="28400-110">下面的示例生成 CS1918：</span><span class="sxs-lookup"><span data-stu-id="28400-110">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
