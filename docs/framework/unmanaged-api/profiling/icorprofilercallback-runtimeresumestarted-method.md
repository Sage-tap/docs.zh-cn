# <a name="icorprofilercallbackruntimeresumestarted-method"></a>ICorProfilerCallback::RuntimeResumeStarted 方法

## <a name="syntax"></a>语法  
  
```cpp  
HRESULT RuntimeResumeStarted();  
```
