# <a name="datamemberlisteditor-class"></a>DataMemberListEditor 类

提供一个下拉用户界面，用于编辑具有非 null 属性的数据绑定对象的属性 (`DataSource`) 通过列出对象的所有属性 `DataSource` ，从中选择值源。  
  
## <a name="syntax"></a>语法
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor`类是内部的，不应在代码中直接使用。
>
> 在任何情况下，Microsoft 不支持在生产应用程序中使用此类。
