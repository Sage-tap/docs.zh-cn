# <a name="compiler-error-cs0588"></a>编译器错误 CS0588

内部编译器错误: 步骤“LEX”

 尝试确定编译器是否因为无法分析意外的语法而产生故障。 如果不是这种情况，请参阅 [Visual Studio 反馈选项](/visualstudio/ide/feedback-options)。

> [!NOTE]
> Roslyn 中不再使用此编译器错误。
