# <a name="compiler-error-cs1563"></a>编译器错误 CS1563

输出“输出文件”没有任何源文件  
  
 已指定 [**OutputAssembly**](../language-reference/compiler-options/output.md#outputassembly) 编译器选项，但后面没有源代码文件。 不应使用 **OutputAssembly**，或者应在 **OutputAssembly** 之后指定源代码文件。
