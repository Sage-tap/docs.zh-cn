# <a name="compiler-error-cs1678"></a><span data-ttu-id="fc0a6-103">编译器错误 CS1678</span><span class="sxs-lookup"><span data-stu-id="fc0a6-103">Compiler Error CS1678</span></span>

<span data-ttu-id="fc0a6-104">参数“number”被声明为类型“type1”，而它应为“type2”</span><span class="sxs-lookup"><span data-stu-id="fc0a6-104">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="fc0a6-105">当匿名方法中的参数类型不同于你要将方法转换为的委托的声明时，将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="fc0a6-105">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="fc0a6-106">下面的示例生成 CS1678：</span><span class="sxs-lookup"><span data-stu-id="fc0a6-106">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
