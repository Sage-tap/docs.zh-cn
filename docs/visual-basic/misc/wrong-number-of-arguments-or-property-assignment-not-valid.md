# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>参数的个数错误或属性赋值无效。

已尝试赋值，但无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查你所提供的参数数目是否与目标所需的数量匹配。  
  
2. 检查属性赋值。
