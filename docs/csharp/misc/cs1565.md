# <a name="compiler-error-cs1565"></a>编译器错误 CS1565

指定的选项冲突: Win32 资源文件；Win32 图标  
  
 在同一编译中同时指定 [**Win32Resource**](../language-reference/compiler-options/resources.md#win32resource) 编译器选项和 [**Win32Icon**](../language-reference/compiler-options/resources.md#win32icon) 编译器选项是无效的。
