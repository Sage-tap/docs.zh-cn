<span data-ttu-id="d0a3e-104">多个嵌套循环由单个 `Next` 语句终止，并且该语句指定的循环变量多于嵌套中的循环数，如下例所示：</span><span class="sxs-lookup"><span data-stu-id="d0a3e-104">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

   For J = 1 To 5  
      ' ...
