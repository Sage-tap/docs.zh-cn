# <a name="unable-to-parse-xml-error"></a>无法分析 XML：\<error>

正在使用的 XML 文件中有错误。  
  
 **错误 ID:** BC31023  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证 XML 格式是否正确有效，并且该文件未损坏。
