# <a name="object-doesnt-support-named-arguments"></a>对象不支持已命名的自变量

你尝试访问其方法不支持已命名的参数的对象。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 访问此对象上的方法时，按位置指定参数。 有关参数位置和类型的详细信息，请参阅对象的文档。
