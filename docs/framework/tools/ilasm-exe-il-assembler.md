<span data-ttu-id="5eaae-228">IL 汇编程序有助于工具供应商设计和实现 IL 生成器。</span><span class="sxs-lookup"><span data-stu-id="5eaae-228">The IL Assembler helps tool vendors design and implement IL generators.</span></span> <span data-ttu-id="5eaae-229">通过使用 Ilasm.exe，工具和编译器开发人员可以专注于生成 IL 和元数据，而无需考虑如何以 PE 文件格式发出 IL。</span><span class="sxs-lookup"><span data-stu-id="5eaae-229">Using *Ilasm.exe*, tool and compiler developers can concentrate on IL and metadata generation without being concerned with emitting IL in the PE file format.</span></span>
