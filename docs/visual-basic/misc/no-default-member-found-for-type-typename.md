# <a name="no-default-member-found-for-type-typename"></a>找不到类型 "" 的默认成员 \<typename>

已尝试后期绑定索引 `Get` 或 `Set` 调用。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保该对象具有带 `Get` 或 `Set`的默认属性，具体取决于你试图进行的操作。
