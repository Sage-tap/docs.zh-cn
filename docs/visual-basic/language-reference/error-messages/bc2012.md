# <a name="bc2012-cant-open-filename-for-writing"></a>BC2012：无法打开 " \<filename> " 进行写入

无法打开指定的文件进行写入，可能是因为它已经被打开。

 **错误 ID：** BC2012

## <a name="to-correct-this-error"></a>更正此错误

1. 关闭并重新打开该文件。

2. 检查文件的权限。
