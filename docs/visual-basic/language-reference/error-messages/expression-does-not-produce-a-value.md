# <a name="bc30491-expression-does-not-produce-a-value"></a>BC30491：表达式不生成值

您尝试使用的表达式不在生成值的上下文中生成值，如 `Sub` 在所需的上下文中调用 `Function` 。

 **错误 ID：** BC30491

## <a name="to-correct-this-error"></a>更正此错误

- 将表达式更改为一个生成值的表达式。
