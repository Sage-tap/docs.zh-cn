# <a name="icordebugchaingetreason-method"></a>ICorDebugChain::GetReason 方法

获取此调用链的 genesis 的原因。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetReason (  
    [out] CorDebugChainReason *pReason  
);  
```  
  
## <a name="parameters"></a>参数  

 `pReason`  
 弄一个指针 (，它指向 CorDebugChainReason 枚举的按位组合) ，该枚举指示此调用链的的原因。
