# <a name="compiler-error-cs0216"></a><span data-ttu-id="3c194-103">编译器错误 CS0216</span><span class="sxs-lookup"><span data-stu-id="3c194-103">Compiler Error CS0216</span></span>

<span data-ttu-id="3c194-104">运算符“operator”要求也要定义匹配的运算符“missing_operator”</span><span class="sxs-lookup"><span data-stu-id="3c194-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="3c194-105">用户定义的 [==](../language-reference/operators/equality-operators.md#equality-operator-) 运算符需要用户定义的 [！ =](../language-reference/operators/equality-operators.md#inequality-operator-) 运算符，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="3c194-105">A user-defined [==](../language-reference/operators/equality-operators.md#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/equality-operators.md#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="3c194-106">这同样适用于用户定义的 [true](../language-reference/operators/true-false-operators.md) 运算符和用户定义的 [false](../language-reference/operators/true-false-operators.md) 运算符。</span><span class="sxs-lookup"><span data-stu-id="3c194-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="3c194-107">下面的示例生成 CS0216：</span><span class="sxs-lookup"><span data-stu-id="3c194-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
