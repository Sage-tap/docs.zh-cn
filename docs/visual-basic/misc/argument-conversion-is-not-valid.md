# <a name="argument-conversion-is-not-valid"></a>参数“Conversion”无效

`VbStrConv` 枚举的值无效，可能因为它小于零。 枚举值必须大于或等于零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保该值大于或等于零。
