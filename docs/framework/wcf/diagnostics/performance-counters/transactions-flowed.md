# <a name="transactions-flowed"></a>Transactions Flowed（流动的事务数）

计数器名称：Transactions Flowed（流动的事务数）  
  
## <a name="description"></a>说明  

 流向此操作的事务数。 只要发送到操作的消息中出现事务 ID，该计数器就会增加。
