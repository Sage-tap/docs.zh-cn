## <a name="example"></a><span data-ttu-id="cbf6d-109">示例</span><span class="sxs-lookup"><span data-stu-id="cbf6d-109">Example</span></span>  

 <span data-ttu-id="cbf6d-110">下面的示例生成 CS3018。</span><span class="sxs-lookup"><span data-stu-id="cbf6d-110">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
