# <a name="compiler-error-cs0040"></a>编译器错误 CS0040

创建调试信息文件时出现意外错误—“原因”  
  
 使用 [**DebugType**](../language-reference/compiler-options/code-generation.md#debugtype) 编译器选项时，如果编译器无法写入 .pdb 文件，则会发生此错误。 针对此错误的可能解决方法包括重新安装 Visual Studio、确保编译器具有写入文件或目录的访问权限，或者不使用 /debug 进行编译。
