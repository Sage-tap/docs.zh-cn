# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>“Throw”语句在即时窗口中无效

`Throw` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30725  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `Throw` 语句。
