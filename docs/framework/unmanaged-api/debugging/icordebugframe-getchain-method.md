# <a name="icordebugframegetchain-method"></a>ICorDebugFrame::GetChain 方法

获取一个指针，该指针指向此帧所属的链。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetChain (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>参数  

 `ppChain`  
 弄指向 ICorDebugChain 对象的地址的指针，该对象表示包含此帧的链。
