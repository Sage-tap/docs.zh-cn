# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>参数“\<argumentname>”的类型为“\<typename>”，它不是数值类型

在需要数值参数的情况下出现了非数值参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保参数是数值。
