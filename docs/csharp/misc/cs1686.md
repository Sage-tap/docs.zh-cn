## <a name="example"></a><span data-ttu-id="7460b-106">示例</span><span class="sxs-lookup"><span data-stu-id="7460b-106">Example</span></span>  

 <span data-ttu-id="7460b-107">下面的示例生成 CS1686。</span><span class="sxs-lookup"><span data-stu-id="7460b-107">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
