# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>"" \<implementsclause> 无法实现 " \<typename> "，因为 " \<typename> " 是保留名称

`Implements` 语句指定了保留的名称。  
  
 **错误 ID：** BC31415  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 重命名无法实现的接口成员。
