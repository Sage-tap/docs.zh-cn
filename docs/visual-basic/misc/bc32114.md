# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="dd4d4-103">具有“Structure”约束的类型形参不能用作约束</span><span class="sxs-lookup"><span data-stu-id="dd4d4-103">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>

<span data-ttu-id="dd4d4-104">具有 `Structure` 约束的类型形参被用作另一个类型形参的约束。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-104">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="dd4d4-105">`Structure` 约束要求传递给其类型形参的类型实参为值类型。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-105">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="dd4d4-106">但是，值类型是无法实现或继承的，因此用它作为约束毫无意义，这种操作将需要其他类型形参实现或继承它。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-106">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="dd4d4-107">只有当两个类型形参的类型完全相同时，才能有意义地解释这种情况。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-107">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="dd4d4-108">如果是这样，则泛型类型仅需要一个类型形参。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-108">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="dd4d4-109">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-109">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="dd4d4-110">传递给 `t2` 的类型无法实现或继承传递给 `t1`的类型，因为传递给 `t1` 的类型必须是值类型。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-110">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="dd4d4-111">**错误 ID：** BC32114</span><span class="sxs-lookup"><span data-stu-id="dd4d4-111">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="dd4d4-112">更正此错误</span><span class="sxs-lookup"><span data-stu-id="dd4d4-112">To correct this error</span></span>  
  
- <span data-ttu-id="dd4d4-113">从另一个类型形参的约束列表中删除约束为 `Structure` 的类型形参。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-113">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="dd4d4-114">如果两个类型形参需要相同的值类型，请仅使用一个类型形参定义泛型类型。</span><span class="sxs-lookup"><span data-stu-id="dd4d4-114">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
