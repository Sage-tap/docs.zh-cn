## <a name="to-correct-this-error"></a><span data-ttu-id="39301-114">更正此错误</span><span class="sxs-lookup"><span data-stu-id="39301-114">To correct this error</span></span>  
  
- <span data-ttu-id="39301-115">如果想让过程通过 `ByRef` 参数为变量赋值而变量是否已持有值并不重要，则无需进行任何操作。</span><span class="sxs-lookup"><span data-stu-id="39301-115">If you intend the procedure to assign a value to the variable through the `ByRef` argument, and if it does not matter whether the variable already holds a value, then no action is necessary.</span></span>
