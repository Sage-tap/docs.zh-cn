# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 方法

设置指定方法的相对虚拟地址。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## <a name="parameters"></a>参数  

 `md`  
 中目标方法或方法实现的标记。  
  
 `ulRVA`  
 中代码或数据区域的地址。
