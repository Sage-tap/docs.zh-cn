## <a name="example"></a><span data-ttu-id="c0350-109">示例</span><span class="sxs-lookup"><span data-stu-id="c0350-109">Example</span></span>

 <span data-ttu-id="c0350-110">下面的示例代码生成 CS0650。</span><span class="sxs-lookup"><span data-stu-id="c0350-110">The following example code generates CS0650.</span></span>  
  
```csharp  
// CS0650.cs  
public class MyClass  
{  
   public static void Main()  
   {  
// Generates CS0650. Incorrect array declaration syntax:  
      int myarray[2];
  
      // Correct declaration.  
      int[] myarray2;  
  
      // Declaration and initialization in one statement  
      int[] myArray3= new int[2] {1,2}  
  
      // Access an array element.  
      myarray3[0] = 0;  
    }  
}  
```  
  
 <span data-ttu-id="c0350-111">下面的示例演示创建固定大小的缓冲区时如何使用 `fixed` 关键字：</span><span class="sxs-lookup"><span data-stu-id="c0350-111">The following example shows how to use the `fixed` keyword when you create a fixed size buffer:</span></span>  
  
```csharp  
// This code must appear in an unsafe block.
public struct MyArray
{  
    public fixed char pathName[128];  
}  
```
