# <a name="isymunmanageddispose-interface"></a>ISymUnmanagedDispose 接口

释放非托管资源。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[Destroy 方法](isymunmanageddispose-destroy-method.md)|使基础对象释放所有内部引用，并在所有后续方法调用中返回失败。|  
  
## <a name="requirements"></a>要求  

 **标头：** CorSym，CorSym
