# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed

ID：108  
  
 严重性：错误  
  
 类别：跟踪  
  
## <a name="description"></a>说明  

 此事件指示没有正在记录的已知 PII。 不允许记录已知的 PII。 若要允许已知 PII 的日志记录，请将 "enableLoggingKnownPii" 设置为 `true` Machine.config 中的。此事件将列出进程名称和进程 ID。
