# <a name="parameter-specifier-is-duplicated"></a>参数说明符重复

参数说明符不止一次应用于该参数。  
  
 **错误 ID：** BC30785  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除重复的说明符。
