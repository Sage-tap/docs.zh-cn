### <a name="example"></a><span data-ttu-id="0b747-129">示例</span><span class="sxs-lookup"><span data-stu-id="0b747-129">Example</span></span>

```csharp

// dotnet_style_prefer_auto_properties = false

{
    get

        return age;
    }
}
```
