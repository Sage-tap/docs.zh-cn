# <a name="compiler-error-cs0193"></a><span data-ttu-id="05c7d-103">编译器错误 CS0193</span><span class="sxs-lookup"><span data-stu-id="05c7d-103">Compiler Error CS0193</span></span>

<span data-ttu-id="05c7d-104">\*或-> 运算符必须应用于指针</span><span class="sxs-lookup"><span data-stu-id="05c7d-104">The \* or -> operator must be applied to a pointer</span></span>  
  
<span data-ttu-id="05c7d-105">\*或-> 运算符与指针类型一起使用。</span><span class="sxs-lookup"><span data-stu-id="05c7d-105">The \* or -> operator was used with a nonpointer type.</span></span> <span data-ttu-id="05c7d-106">有关详细信息，请参阅[指针类型](../language-reference/unsafe-code.md#pointer-types)。</span><span class="sxs-lookup"><span data-stu-id="05c7d-106">For more information, see [Pointer types](../language-reference/unsafe-code.md#pointer-types).</span></span>  
  
 <span data-ttu-id="05c7d-107">下面的示例生成 CS0193：</span><span class="sxs-lookup"><span data-stu-id="05c7d-107">The following sample generates CS0193:</span></span>  
  
```csharp  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
