# <a name="referenced-array-element-has-a-value-of-nothing"></a>引用的数组元素的值为“Nothing”

尚未设置所引用的数组元素，或之前被设置为 `Nothing`。  
  
 **错误 ID:** BC30744  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证数组中的值是否正确设置。
