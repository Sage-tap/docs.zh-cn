# <a name="set-not-supported-at-run-time"></a>运行时不支持设置。

你尝试设置或更改其值只能在设计时设置的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 从代码中删除对该属性的引用。  
  
2. 更改引用以便在运行时仅返回该属性的值。
