## <a name="example"></a><span data-ttu-id="ca661-106">示例</span><span class="sxs-lookup"><span data-stu-id="ca661-106">Example</span></span>  

 <span data-ttu-id="ca661-107">以下示例生成 CS1909。</span><span class="sxs-lookup"><span data-stu-id="ca661-107">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
