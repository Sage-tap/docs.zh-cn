## <a name="prepare-the-raspberry-pi"></a>准备 Raspberry Pi

使用 `raspi-config` 命令确保启用以下两个服务：

- SSH
- SPI

有关 `raspi-config` 的详细信息，请参阅 [Raspberry Pi 文档](https://www.raspberrypi.org/documentation/configuration/raspi-config.md)。
