# <a name="-expected-declaration"></a>需要 "=" (声明)

`Mid` 语句或 `#Const` 编译指令不包含赋值。  
  
 **错误 ID：** BC30249  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 为字符串或编译常量添加一个等号 (`=`) 和一个要赋予的值。
