# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>在自定义日志名称中，只有前八个字符是有意义的

在检查事件日志名称的唯一性时，仅考虑前八个字符。 当事件日志共享前八个字符时，则可能会导致冲突。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 为事件日志命名，其中前八个字符是唯一的。
