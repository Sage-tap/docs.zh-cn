# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp

Id：160  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>说明  

 此事件表示在尝试侦听指定站点时指定协议的指定激活服务出错，因此将暂时为该站点禁用该协议。 此事件将列出 URL、状态、异常、进程名称和进程 ID。
