# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>"" \<membername> 已由 \<procedurename> 为此生成的 "" 声明 \<containername>

声明的成员与公共语言运行时声明的某个成员具有相同的名称。

**错误 ID：** BC30733

## <a name="to-correct-this-error"></a>更正此错误

- 更改成员名称以消除冲突。
