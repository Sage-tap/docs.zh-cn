# <a name="compiler-error-cs0254"></a><span data-ttu-id="acbcc-103">编译器错误 CS0254</span><span class="sxs-lookup"><span data-stu-id="acbcc-103">Compiler Error CS0254</span></span>

<span data-ttu-id="acbcc-104">fixed 语句赋值的右边不能是强制转换表达式</span><span class="sxs-lookup"><span data-stu-id="acbcc-104">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="acbcc-105">不能强制转换 [固定](../language-reference/keywords/fixed-statement.md) 表达式的右侧。</span><span class="sxs-lookup"><span data-stu-id="acbcc-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="acbcc-106">有关详细信息，请参阅[不安全代码和指针](../language-reference/unsafe-code.md)。</span><span class="sxs-lookup"><span data-stu-id="acbcc-106">For more information, see [Unsafe Code and Pointers](../language-reference/unsafe-code.md).</span></span>  
  
 <span data-ttu-id="acbcc-107">下面的示例生成 CS0254：</span><span class="sxs-lookup"><span data-stu-id="acbcc-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
