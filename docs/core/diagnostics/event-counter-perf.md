## <a name="add-an-action-filter"></a><span data-ttu-id="11d80-132">添加操作筛选器</span><span class="sxs-lookup"><span data-stu-id="11d80-132">Add an action filter</span></span>

<span data-ttu-id="11d80-133">示例源代码是 ASP.NET Core 项目。</span><span class="sxs-lookup"><span data-stu-id="11d80-133">The sample source code is an ASP.NET Core project.</span></span> <span data-ttu-id="11d80-134">可以全局添加将记录总请求时间的[操作筛选器](/aspnet/core/mvc/controllers/filters#action-filters)。</span><span class="sxs-lookup"><span data-stu-id="11d80-134">You can add an [action filter](/aspnet/core/mvc/controllers/filters#action-filters) globally that will log the total request time.</span></span> <span data-ttu-id="11d80-135">创建名为 LogRequestTimeFilterAttribute.cs 的新文件，并使用以下代码：</span><span class="sxs-lookup"><span data-stu-id="11d80-135">Create a new file named *LogRequestTimeFilterAttribute.cs*, and use the following code:</span></span>

```csharp
using System.Diagnostics;
using Microsoft.AspNetCore.Http.Extensions;
using Microsoft.AspNetCore.Mvc.Filters;

namespace DiagnosticScenarios
{
    public class LogRequestTimeFilterAttribute : ActionFilterAttribute
    {
        readonly Stopwatch _stopwatch = new Stopwatch();

        public override void OnActionExecuting(ActionExecutingContext context) => _stopwatch.Start();

        public override void OnActionExecuted(ActionExecutedContext context)
        {
            _stopwatch.Stop();

            MinimalEventCounterSource.Log.Request(
                context.HttpContext.Request.GetDisplayUrl(), _stopwatch.ElapsedMilliseconds);
        }
    }
}
```

<span data-ttu-id="11d80-136">操作筛选器在请求开始时启动 <xref:System.Diagnostics.Stopwatch>，并在其完成后停止，捕获运行时间。</span><span class="sxs-lookup"><span data-stu-id="11d80-136">The action filter starts a <xref:System.Diagnostics.Stopwatch> as the request begins, and stops after it has completed, capturing the elapsed time.</span></span> <span data-ttu-id="11d80-137">总毫秒数记录到 `MinimalEventCounterSource` 单一实例。</span><span class="sxs-lookup"><span data-stu-id="11d80-137">The total milliseconds is logged to the `MinimalEventCounterSource` singleton instance.</span></span> <span data-ttu-id="11d80-138">为了应用此筛选器，需要将其添加到筛选器集合。</span><span class="sxs-lookup"><span data-stu-id="11d80-138">In order for this filter to be applied, you need to add it to the filters collection.</span></span> <span data-ttu-id="11d80-139">在 Startup.cs 文件中，更新包含此筛选器的 `ConfigureServices` 方法。</span><span class="sxs-lookup"><span data-stu-id="11d80-139">In the *Startup.cs* file, update the `ConfigureServices` method in include this filter.</span></span>

```csharp

    services.AddControllers(options => options.Filters.Add<LogRequestTimeFilterAttribute>())
            .AddNewtonsoftJson();
```
