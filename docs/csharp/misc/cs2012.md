# <a name="compiler-error-cs2012"></a>编译器错误 CS2012

无法打开“file”进行写入  
  
使用 `-bugreport:` `file` 编译器选项时，无法打开文件进行写入。 请确保指定的文件名有效，并且该文件不是只读文件。

> [!IMPORTANT]
> `bugreport`不再支持该选项。
