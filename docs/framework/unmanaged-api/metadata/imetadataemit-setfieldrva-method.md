# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法

为指定的标记所引用的字段的相对虚拟地址设置全局变量值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetFieldRVA (
    [in]  mdFieldDef  fd,
    [in]  ULONG       ulRVA
);  
```  
  
## <a name="parameters"></a>参数  

 `fd`  
 中目标字段的标记。  
  
 `ulRVA`  
 中代码或数据区域的地址。
