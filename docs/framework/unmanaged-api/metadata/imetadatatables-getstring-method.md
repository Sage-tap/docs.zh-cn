# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法

从当前引用范围内的表列中获取指定索引处的字符串。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetString (
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>参数  

 `ixString`  
 中开始搜索下一个值的索引位置。  
  
 `ppString`  
 弄指向返回的字符串值的指针的指针。
