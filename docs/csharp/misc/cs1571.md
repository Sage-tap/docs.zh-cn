# <a name="compiler-warning-level-2-cs1571"></a>编译器警告（等级 2）CS1571

“构造”上的 XML 注释中有重复的“参数”param 标记  
  
 使用 [**DocumentationFile**](../language-reference/compiler-options/output.md#documentationfile) 编译器选项时，找到了同一方法参数的多个注释。 删除其中一个重复行。  
  
 下面的示例生成 CS1571：  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
