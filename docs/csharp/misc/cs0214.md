# <a name="compiler-error-cs0214"></a><span data-ttu-id="78eb0-103">编译器错误 CS0214</span><span class="sxs-lookup"><span data-stu-id="78eb0-103">Compiler Error CS0214</span></span>

<span data-ttu-id="78eb0-104">指针和固定大小缓冲区只能在不安全的上下文中使用</span><span class="sxs-lookup"><span data-stu-id="78eb0-104">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="78eb0-105">指针只能通过 [unsafe](../language-reference/keywords/unsafe.md) 关键字使用。</span><span class="sxs-lookup"><span data-stu-id="78eb0-105">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="78eb0-106">有关详细信息，请参阅[不安全代码和指针](../language-reference/unsafe-code.md)。</span><span class="sxs-lookup"><span data-stu-id="78eb0-106">For more information, see [Unsafe Code and Pointers](../language-reference/unsafe-code.md).</span></span>  
  
 <span data-ttu-id="78eb0-107">下面的示例生成 CS0214：</span><span class="sxs-lookup"><span data-stu-id="78eb0-107">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
