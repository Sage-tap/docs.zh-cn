# <a name="coiniticor-enumeration"></a>COINITICOR 枚举

指定 [CoInitializeCor](../hosting/coinitializecor-function.md) 在初始化公共语言运行时时使用的常量。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```  
  
## <a name="members"></a>成员  
  
|成员|说明|  
|------------|-----------------|  
|`COINITCOR`|指示默认初始化模式。|
