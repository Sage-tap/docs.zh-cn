# <a name="basic-http-service"></a><span data-ttu-id="8ad4a-103">基本 HTTP 服务</span><span class="sxs-lookup"><span data-stu-id="8ad4a-103">Basic HTTP Service</span></span>

<span data-ttu-id="8ad4a-104">此示例演示如何使用 Windows Communication Foundation (WCF) REST 编程模型，实现一般称为 "POX" (纯旧 XML) service）的基于 HTTP 的 RPC 服务。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-104">This sample demonstrates how to implement an HTTP-based, RPC-based service - popularly referred to as "POX" (Plain Old XML) service – using the Windows Communication Foundation (WCF) REST Programming model.</span></span> <span data-ttu-id="8ad4a-105">此示例由两个组件组成：一个自承载 WCF HTTP 服务 (Service.cs) 和一个控制台应用程序)  (用于创建服务并对其进行调用的控制台应用程序。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-105">This sample consists of two components: a self-hosted WCF HTTP service (Service.cs) and a console application (Program.cs) that creates the service and makes calls to it.</span></span>

## <a name="sample-details"></a><span data-ttu-id="8ad4a-106">示例详细信息</span><span class="sxs-lookup"><span data-stu-id="8ad4a-106">Sample Details</span></span>

<span data-ttu-id="8ad4a-107">WCF 服务公开2个操作， `EchoWithGet` 并公开 `EchoWithPost` 作为输入传递的字符串。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-107">The WCF service exposes 2 operations, `EchoWithGet` and `EchoWithPost`, which returns the string that was passed as input.</span></span>

<span data-ttu-id="8ad4a-108">`EchoWithGet` 操作使用 <xref:System.ServiceModel.Web.WebGetAttribute> 添加批注，后者指示该操作处理 HTTP `GET` 请求。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-108">The `EchoWithGet` operation is annotated with <xref:System.ServiceModel.Web.WebGetAttribute>, which indicates that the operation processes HTTP `GET` requests.</span></span> <span data-ttu-id="8ad4a-109">由于 <xref:System.ServiceModel.Web.WebGetAttribute> 不显式指定 <xref:System.UriTemplate>，因此该操作需要使用名称为 `s` 的查询字符串参数来传递输入字符串。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-109">Because the <xref:System.ServiceModel.Web.WebGetAttribute> does not explicitly specify a <xref:System.UriTemplate>, the operation expects the input string to be passed in using a query string parameter with name `s`.</span></span> <span data-ttu-id="8ad4a-110">请注意，可以使用 <xref:System.ServiceModel.Web.WebGetAttribute.UriTemplate%2A> 属性来自定义服务所需的 URI 格式。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-110">Note that the format of the URI that the service expects can be customized using the <xref:System.ServiceModel.Web.WebGetAttribute.UriTemplate%2A> property.</span></span>

<span data-ttu-id="8ad4a-111">`EchoWithPost` 操作使用 <xref:System.ServiceModel.Web.WebInvokeAttribute> 添加批注，后者指示该操作不是 `GET` 操作（它具有副作用）。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-111">The `EchoWithPost` operation is annotated with <xref:System.ServiceModel.Web.WebInvokeAttribute>, which indicates it is not a `GET` operation (it has side effects).</span></span> <span data-ttu-id="8ad4a-112">由于 <xref:System.ServiceModel.Web.WebInvokeAttribute> 不显式指定 `Method`，因此该操作处理在请求正文中具有字符串（例如，以 XML 格式）的 HTTP `POST` 请求。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-112">Because the <xref:System.ServiceModel.Web.WebInvokeAttribute> does not explicitly specify a `Method`, the operation processes HTTP `POST` requests that have the string in the request body (in the XML format, for instance).</span></span> <span data-ttu-id="8ad4a-113">请注意，可分别使用 <xref:System.ServiceModel.Web.WebInvokeAttribute.Method%2A> 和 <xref:System.ServiceModel.Web.WebInvokeAttribute.UriTemplate> 属性来自定义该请求的 URI 的 HTTP 方法和格式。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-113">Note that the HTTP method and the format of the URI for the request can be customized using the <xref:System.ServiceModel.Web.WebInvokeAttribute.Method%2A> and <xref:System.ServiceModel.Web.WebInvokeAttribute.UriTemplate> properties respectively.</span></span>

<span data-ttu-id="8ad4a-114">App.config 文件使用 <xref:System.ServiceModel.Description.WebHttpEndpoint> 属性设置为 <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> 的默认 `true` 来配置 WCF 服务。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-114">The App.config file configures the WCF service with a default <xref:System.ServiceModel.Description.WebHttpEndpoint> that has the <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> property set to `true`.</span></span> <span data-ttu-id="8ad4a-115">因此，WCF 基础结构将在中创建一个基于 HTML 的自动帮助页 `http://localhost:8000/Customers/help` ，该帮助页提供有关如何构造对服务的 http 请求以及如何使用该服务的 HTTP 响应的信息。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-115">As a result, the WCF infrastructure creates an automatic HTML based help page at `http://localhost:8000/Customers/help` that provides information about how to construct HTTP requests to the service and how to consume the service’s HTTP response.</span></span>

<span data-ttu-id="8ad4a-116">Program.cs 演示如何使用 WCF 通道工厂对服务进行调用并处理响应。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-116">Program.cs demonstrates how a WCF channel factory can be used to make calls to the service and process responses.</span></span> <span data-ttu-id="8ad4a-117">请注意，这只是访问 WCF 服务的一种方式。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-117">Note that this is just one way to access a WCF service.</span></span> <span data-ttu-id="8ad4a-118">还可以使用其他 .NET Framework 类（如 <xref:System.Net.HttpWebRequest> 和 <xref:System.Net.WebClient>）来访问该服务。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-118">It is also possible to access the service using other .NET Framework classes like <xref:System.Net.HttpWebRequest> and <xref:System.Net.WebClient>.</span></span>

<span data-ttu-id="8ad4a-119">此示例包含一个自承载服务和一个客户端，它们都在一个控制台应用程序内运行。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-119">The sample consists a self-hosted service and a client that both run within a console application.</span></span> <span data-ttu-id="8ad4a-120">在控制台应用程序运行时，客户端会对服务进行请求，并将响应中的相关信息写入控制台窗口。</span><span class="sxs-lookup"><span data-stu-id="8ad4a-120">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>
