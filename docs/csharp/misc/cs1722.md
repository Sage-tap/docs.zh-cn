# <a name="compiler-error-cs1722"></a><span data-ttu-id="4ac41-103">编译器错误 CS1722</span><span class="sxs-lookup"><span data-stu-id="4ac41-103">Compiler Error CS1722</span></span>

## <a name="example"></a><span data-ttu-id="4ac41-106">示例</span><span class="sxs-lookup"><span data-stu-id="4ac41-106">Example</span></span>  

 <span data-ttu-id="4ac41-107">下面的示例生成 CS1722。</span><span class="sxs-lookup"><span data-stu-id="4ac41-107">The following sample generates CS1722.</span></span>  
  
```csharp  
// CS1722.cs  
// compile with: /target:library  
public class A {}  
interface I {}  
  
public class MyClass : I, A {}   // CS1722  
public class MyClass2 : A, I {}   // OK  
```
