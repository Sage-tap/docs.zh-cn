# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed

Id：126  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>说明  

 此事件指示 WMI 提供程序未注册。 此事件将列出 WMI 对象、错误、进程名称和进程 ID。
