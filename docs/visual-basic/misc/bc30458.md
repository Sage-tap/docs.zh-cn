# <a name="types-are-not-available-in-this-context"></a>此上下文中类型不可用

尝试引用 `#If` 指令中的类型。  
  
 **错误 ID:** BC30458  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除引用。
