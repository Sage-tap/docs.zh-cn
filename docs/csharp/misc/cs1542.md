# <a name="compiler-error-cs1542"></a>编译器错误 CS1542

“dll”无法添加到此程序集中，因为它已经是程序集；请使用“/R”选项  
  
 用 [**AddModules**](../language-reference/compiler-options/inputs.md#addmodules)编译器选项引用的文件不是用 [ **TargetType** 编译器选项](../language-reference/compiler-options/output.md#targettype)的 **module** 元素生成的;在此编译 [**中使用引用来引用文件**](../language-reference/compiler-options/inputs.md#references)。
