# <a name="anonymous-type-property-propertyname-cannot-be-used-in-the-definition-of-a-lambda-expression-within-the-same-initialization-list"></a><span data-ttu-id="dcf79-103">不能在同一个初始化列表中的 lambda 表达式定义中使用匿名类型属性“\<propertyname>”</span><span class="sxs-lookup"><span data-stu-id="dcf79-103">Anonymous type property '\<propertyname>' cannot be used in the definition of a lambda expression within the same initialization list</span></span>

<span data-ttu-id="dcf79-104">匿名类型的初始化列表中定义的属性不能是相同列表中的 lambda 表达式定义的一部分。</span><span class="sxs-lookup"><span data-stu-id="dcf79-104">Properties defined in the initialization list of an anonymous type cannot be part of a lambda expression definition in the same list.</span></span> <span data-ttu-id="dcf79-105">例如，在下面的代码中，属性 `Num` 不能包含在 `LambdaFun`的定义中。</span><span class="sxs-lookup"><span data-stu-id="dcf79-105">For example, in the following code, property `Num` cannot be included in the definition of `LambdaFun`.</span></span>

```vb
' Not valid.
'Dim anon = New With {.Num = 4, .LambdaFun = Function() .Num > 0}
```

<span data-ttu-id="dcf79-106">**错误 ID：** BC36549</span><span class="sxs-lookup"><span data-stu-id="dcf79-106">**Error ID:** BC36549</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="dcf79-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="dcf79-107">To correct this error</span></span>

1. <span data-ttu-id="dcf79-108">考虑将匿名类型拆分为两个部分：</span><span class="sxs-lookup"><span data-stu-id="dcf79-108">Consider splitting the anonymous type into two parts:</span></span>

    ```vb
    Dim anon1 = New With {.Num = 4}
    Dim anon2 = New With {.LambdaFun = Function() anon1.Num > 0}
    ' - or -
    Dim anon3 = New With {.lambdaFun = Function(n As Integer) n > 0}
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    anon1.Num = -5
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    ```

    <span data-ttu-id="dcf79-109">请注意，如果将 `anon1.Num` 声明为 `Key` 属性，则无法更改其值。</span><span class="sxs-lookup"><span data-stu-id="dcf79-109">Note that if you declare `anon1.Num` as a `Key` property, its value cannot be changed.</span></span>

2. <span data-ttu-id="dcf79-110">一种替代方法是使用常规函数语句访问匿名类型属性：</span><span class="sxs-lookup"><span data-stu-id="dcf79-110">An alternative is to use a regular function statement to access the anonymous type property:</span></span>

    ```vb
    Function testNum(ByVal n As Integer) As Boolean
        Return n > 0
    End Function
    Console.WriteLine(testNum(anon1.Num))
    ```

3. <span data-ttu-id="dcf79-111">同样，可以使用在匿名类型外部定义的 lambda 函数：</span><span class="sxs-lookup"><span data-stu-id="dcf79-111">Similarly, you can use a lambda function that is defined outside the anonymous type:</span></span>

    ```vb
    Dim lambdaFun1 = Function() anon1.Num > 0
    Dim lambdaFun2 = Function(n As Integer) n > 0
    ```
