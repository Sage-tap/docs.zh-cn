# <a name="compiler-warning-level-1-cs1598"></a>编译器警告（等级 1）CS1598

无法加载 XML 分析程序，原因如下：“reason”。 不会生成 XML 文档文件“file”。  
  
 已指定 [DocumentationFile](../compiler-options/output.md#documentationfile) 选项，但编译器无法找到并加载 msxml3.dll。 请确保已安装并注册文件 msxml3.dll。
