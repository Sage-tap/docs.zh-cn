# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="6d442-103">无法使用匿名类型成员属性“\<propertyname>”来推理另一个成员属性的类型，因为尚未建立“\<propertyname>”的类型</span><span class="sxs-lookup"><span data-stu-id="6d442-103">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="6d442-104">直到建立匿名类型属性的类型，它才可用于建立其他属性的类型。</span><span class="sxs-lookup"><span data-stu-id="6d442-104">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="6d442-105">例如，在下面的声明中， `.IDName = .LastName` 无效，因为尚未初始化 `.LastName` 。</span><span class="sxs-lookup"><span data-stu-id="6d442-105">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="6d442-106">**错误 ID：** BC36559</span><span class="sxs-lookup"><span data-stu-id="6d442-106">**Error ID:** BC36559</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="6d442-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="6d442-107">To correct this error</span></span>

