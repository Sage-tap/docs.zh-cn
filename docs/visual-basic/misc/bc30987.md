# <a name="-expected"></a><span data-ttu-id="50270-103">应有“{”</span><span class="sxs-lookup"><span data-stu-id="50270-103">'{' expected</span></span>

<span data-ttu-id="50270-104">若要使用对象初始值设定项声明命名类型或匿名类型的实例，必须将字段或属性及其初始值列表括在大括号（{ 和 }）中。</span><span class="sxs-lookup"><span data-stu-id="50270-104">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```vb  
Dim client As New Customer() With {.Name = "Microsoft", .City = "Seattle"}  
Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}  
Dim anon = New With {.ID = 123456}  
```
