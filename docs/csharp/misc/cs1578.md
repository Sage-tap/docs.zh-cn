# <a name="compiler-error-cs1578"></a><span data-ttu-id="ad032-103">编译器错误 CS1578</span><span class="sxs-lookup"><span data-stu-id="ad032-103">Compiler Error CS1578</span></span>

<span data-ttu-id="ad032-104">应输入文件名、单行注释或行尾</span><span class="sxs-lookup"><span data-stu-id="ad032-104">Filename, single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="ad032-105">[#line](../language-reference/preprocessor-directives.md#error-and-warning-information) 指令后面只能出现文件名（双引号括起）或单行注释。</span><span class="sxs-lookup"><span data-stu-id="ad032-105">After a [#line](../language-reference/preprocessor-directives.md#error-and-warning-information) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="ad032-106">以下示例生成 CS1578：</span><span class="sxs-lookup"><span data-stu-id="ad032-106">The following sample generates CS1578:</span></span>  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
