<span data-ttu-id="1f71d-104">参数“number”不应使用“keyword”关键字进行声明</span><span class="sxs-lookup"><span data-stu-id="1f71d-104">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="1f71d-105">当匿名方法中的参数类型修饰符不同于你在要将该方法强制转换为的委托的声明中所用的参数类型修饰符时，将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="1f71d-105">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1f71d-106">示例</span><span class="sxs-lookup"><span data-stu-id="1f71d-106">Example</span></span>  

 <span data-ttu-id="1f71d-107">以下示例生成 CS1677：</span><span class="sxs-lookup"><span data-stu-id="1f71d-107">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
