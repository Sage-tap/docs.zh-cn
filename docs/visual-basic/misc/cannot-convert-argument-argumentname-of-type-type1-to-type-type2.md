# <a name="cannot-convert-argument-argumentname-of-type-type1-to-type-type2"></a>无法将类型为 \<argumentname> "" 的参数 "" 转换 \<type1> 为类型 " \<type2> "

转换操作尝试将参数转换为不兼容的类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保你尝试将参数转换到的类型有效。
