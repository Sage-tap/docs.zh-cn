# <a name="argument-argumentname-must-be-greater-than-0-or-equal-to--1"></a>参数“\<argumentname>”必须大于 0 或等于 -1

参数的值小于 0，但不等于 -1。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保参数的值在有效范围内。  
  
2. 如果参数由某个计算提供，则检查该计算所处理的值。
