# <a name="_corimageunloading-function"></a>_CorImageUnloading 函数

卸载托管模块映像时通知加载程序。  
  
 未实现此函数。 如果调用，它将返回 E_NOTIMPL。  
  
## <a name="syntax"></a>语法  
  
```cpp  
STDAPI (VOID) _CorImageUnloading(
   [in] PVOID* ImageBase  
);  
```  
  
## <a name="parameters"></a>参数  

 `ImageBase`  
 中一个指针，指向要卸载的图像的起始位置。
