# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile 方法

将指定的文件映射到内存。 此方法已过时。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
## <a name="parameters"></a>参数  

 `hFile`  
 中要映射的文件的句柄。  
  
 `hMapAddress`  
 弄开始映射文件的起始内存地址。
