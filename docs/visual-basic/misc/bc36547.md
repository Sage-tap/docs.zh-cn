# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="08ddd-103">已声明了匿名类型成员或属性“\<propertyname>”</span><span class="sxs-lookup"><span data-stu-id="08ddd-103">Anonymous type member or property '\<propertyname>' is already declared</span></span>

<span data-ttu-id="08ddd-104">属性名在匿名类型的声明中只能建立一次。</span><span class="sxs-lookup"><span data-stu-id="08ddd-104">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="08ddd-105">例如，下面的声明无效：</span><span class="sxs-lookup"><span data-stu-id="08ddd-105">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="08ddd-106">**错误 ID：** BC36547</span><span class="sxs-lookup"><span data-stu-id="08ddd-106">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="08ddd-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="08ddd-107">To correct this error</span></span>  
  
- <span data-ttu-id="08ddd-108">为其中一个属性选择其他名称。</span><span class="sxs-lookup"><span data-stu-id="08ddd-108">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="08ddd-109">为你将从中推断名称和值的变量或属性提供新名称。</span><span class="sxs-lookup"><span data-stu-id="08ddd-109">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
