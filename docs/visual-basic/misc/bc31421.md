# <a name="member-is-already-declared-in-this-structure"></a>\<member>已在此结构中声明了 ""

已多次在此结构中声明枚举成员。  
  
 **错误 ID：** BC31421  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除无关联的声明。
