# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>不能将属性 \<propertyname> 设置为空字符串或 Nothing

该属性要求使用除空字符串 ("") 或 `Nothing`以外的值。 如果计算值时有误，则可能会出现此结果。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 提供一个有效值。
