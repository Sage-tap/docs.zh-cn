# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete

Id：150  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## <a name="description"></a>说明  

 此事件指示 WS-Atomic Transaction 协议服务已成功完成启动和恢复。 此事件将列出协议 ID、协议名称、进程名称和进程 ID。
