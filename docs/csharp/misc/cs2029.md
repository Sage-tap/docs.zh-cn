# <a name="compiler-warning-level-1-cs2029"></a>编译器警告（等级 1）CS2029

“/define”的值无效；“identifier”不是有效的标识符  
  
 如果在 [**保留 defineconstants**](../language-reference/compiler-options/language.md#defineconstants) 选项中使用的值具有一些无效字符，则会出现此警告。  
  
 [**DisabledWarnings**](../language-reference/compiler-options/errors-warnings.md#disabledwarnings)选项无法禁止显示此警告。
