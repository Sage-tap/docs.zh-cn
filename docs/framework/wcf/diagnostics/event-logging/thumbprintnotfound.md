# <a name="thumbprintnotfound"></a>ThumbPrintNotFound

Id：152  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## <a name="description"></a>说明  

 此事件指示无法找到带有特定指纹的标识证书。 事件将列出进程名称和进程 ID。
