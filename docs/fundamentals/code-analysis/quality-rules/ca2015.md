## <a name="how-to-fix-violations"></a><span data-ttu-id="949bc-117">如何解决冲突</span><span class="sxs-lookup"><span data-stu-id="949bc-117">How to fix violations</span></span>

<span data-ttu-id="949bc-118">若要解决此冲突，请删除终结器定义。</span><span class="sxs-lookup"><span data-stu-id="949bc-118">To fix the violation, remove the finalizer definition.</span></span>

```csharp
class DerivedClass <T> : MemoryManager<T>
{
    public override bool Dispose(bool disposing)
    {
        if (disposing)
        {
            _handle.Dispose();
        }
    }

    ...

    // Violation occurs, remove the finalizer to fix the warning.
    ~DerivedClass() => Dispose(false);
}
```
