# <a name="lib-expected"></a>需要 "Lib"

`Declare` 语句指定某个库名时没有使用 `Lib` 关键字。  
  
 **错误 ID：** BC30218  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在库名前添加 `Lib` 关键字。
