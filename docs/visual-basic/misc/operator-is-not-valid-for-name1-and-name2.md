# <a name="operator-is-not-valid-for-name1-and-name2"></a>运算符对于 " \<name1> " 和 " \<name2> " 无效

运算符不适合于正运算的类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保该运算符适合于正运算的类型。
