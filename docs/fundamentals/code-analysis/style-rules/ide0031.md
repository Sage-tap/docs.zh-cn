### <a name="example"></a><span data-ttu-id="8f29a-127">示例</span><span class="sxs-lookup"><span data-stu-id="8f29a-127">Example</span></span>

```csharp
// dotnet_style_null_propagation = true

// dotnet_style_null_propagation = false

```

