# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken 方法

获取 `TypeDef` 引用此类的定义的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
## <a name="parameters"></a>参数  

 `pTypeDef`  
 弄指向 `mdTypeDef` 引用此类的定义的标记的指针。
