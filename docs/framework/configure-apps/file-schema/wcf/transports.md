# <a name="transports"></a>传输

传输层是通道堆栈的最低层。 Windows Communication Foundation 中使用的主要传输 (WCF) 是 HTTP、HTTPS、TCP 和命名管道。 本节列出了可用以配置不同类型的传输的所有配置元素。
