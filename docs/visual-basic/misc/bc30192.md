# <a name="end-of-parameter-list-expected"></a>预期的参数列表末尾

在过程声明中， `ParamArray` 规范后跟一个或多个其他参数规范。  
  
 **错误 ID：** BC30192  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `ParamArray` 移到参数列表的末尾。
