## <a name="example"></a><span data-ttu-id="da2b5-130">示例</span><span class="sxs-lookup"><span data-stu-id="da2b5-130">Example</span></span>

<span data-ttu-id="da2b5-131">在下面的代码中， `example1` 字符串违反规则 CA1305。</span><span class="sxs-lookup"><span data-stu-id="da2b5-131">In the following code, the `example1` string violates rule CA1305.</span></span> <span data-ttu-id="da2b5-132">`example2`字符串满足规则 CA1305，方法是 <xref:System.Globalization.CultureInfo.CurrentCulture%2A?displayProperty=nameWithType> 将实现的传递 <xref:System.IFormatProvider> 到 <xref:System.String.Format(System.IFormatProvider,System.String,System.Object)?displayProperty=nameWithType> 。</span><span class="sxs-lookup"><span data-stu-id="da2b5-132">The `example2` string satisfies rule CA1305 by passing <xref:System.Globalization.CultureInfo.CurrentCulture%2A?displayProperty=nameWithType>, which implements <xref:System.IFormatProvider>, to <xref:System.String.Format(System.IFormatProvider,System.String,System.Object)?displayProperty=nameWithType>.</span></span> <span data-ttu-id="da2b5-133">`example3`通过将内插字符串传递到，字符串满足规则 CA1305 <xref:System.FormattableString.Invariant%2A?displayProperty=fullName]> 。</span><span class="sxs-lookup"><span data-stu-id="da2b5-133">The `example3` string satisfies rule CA1305 by passing an interpolated string to <xref:System.FormattableString.Invariant%2A?displayProperty=fullName]>.</span></span>

```csharp
string name = "Georgette";

// Violates CA1305
string example1 = String.Format("Hello {0}", name);

// Satisfies CA1305
string example2 = String.Format(CultureInfo.CurrentCulture, "Hello {0}", name);

// Satisfies CA1305
string example3 = FormattableString.Invariant($"Hello {name}");
```
