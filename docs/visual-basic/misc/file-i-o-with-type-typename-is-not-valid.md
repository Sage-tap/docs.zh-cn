# <a name="file-io-with-type-typename-is-not-valid"></a>类型为 "" 的文件 i/o \<typename> 无效

尝试将不受支持的类型写入文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
