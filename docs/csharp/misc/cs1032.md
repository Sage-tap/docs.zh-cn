 <span data-ttu-id="aa943-105">`#define` 和 `#undef` [预处理器指令](../language-reference/preprocessor-directives.md#defining-symbols) 必须用在程序的开头，并且在其他任何关键字之前，例如用在命名空间声明中的那些指令之前。</span><span class="sxs-lookup"><span data-stu-id="aa943-105">The `#define` and `#undef` [preprocessor directives](../language-reference/preprocessor-directives.md#defining-symbols) must be used at the beginning of a program, before any other keywords, such as those used in the namespace declaration.</span></span>  
  
 <span data-ttu-id="aa943-106">下面的示例生成 CS1032：</span><span class="sxs-lookup"><span data-stu-id="aa943-106">The following sample generates CS1032:</span></span>

```csharp  
   {  
      #define a   // CS1032, put before namespace

   }  
}  
```
