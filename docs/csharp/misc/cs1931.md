## <a name="example"></a><span data-ttu-id="22cba-108">示例</span><span class="sxs-lookup"><span data-stu-id="22cba-108">Example</span></span>  

 <span data-ttu-id="22cba-109">下面的代码生成 CS1931，因为该标识符 `x` 既用作 `Main` 中的本地变量，也用作查询表达式中的范围变量：</span><span class="sxs-lookup"><span data-stu-id="22cba-109">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

```csharp  
    {  
        static void Main()

            int x = 1;

                    select x;  
        }  
    }  
```
