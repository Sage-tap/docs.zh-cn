## <a name="example"></a><span data-ttu-id="bf369-109">示例</span><span class="sxs-lookup"><span data-stu-id="bf369-109">Example</span></span>  

 <span data-ttu-id="bf369-110">以下示例生成 CS3021：</span><span class="sxs-lookup"><span data-stu-id="bf369-110">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
