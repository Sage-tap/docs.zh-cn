# <a name="add-failed-duplicate-key-value-supplied"></a>添加失败。 提供了重复的键值

`Add` 操作失败，因为该键值等同于另一个键值。 键值必须是唯一的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保键值是唯一的。
