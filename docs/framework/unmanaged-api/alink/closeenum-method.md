# <a name="closeenum-method"></a>CloseEnum 方法

关闭所指示的枚举并释放关联的资源。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
## <a name="parameters"></a>参数  

 `hEnum`  
 要关闭的枚举的句柄。  
  
## <a name="return-value"></a>返回值  

 如果方法成功，则返回 S_OK。  
  
## <a name="requirements"></a>要求  

 需要 alink
