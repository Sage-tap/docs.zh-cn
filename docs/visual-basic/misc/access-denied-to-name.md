# <a name="access-denied-to-name"></a>拒绝访问 " \<name> "

因为计算机安全权限不足，设备、文件或磁盘不可访问。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保你有必要的权限。
