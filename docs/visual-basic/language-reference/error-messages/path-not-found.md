# <a name="path-not-found"></a>找不到路径

在文件访问或磁盘访问操作期间，操作系统找不到指定的路径。 文件的路径包含驱动器规范以及必须遍历以定位文件的目录和子目录。 路径可以是相对路径或绝对路径。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证并重新指定该路径。
