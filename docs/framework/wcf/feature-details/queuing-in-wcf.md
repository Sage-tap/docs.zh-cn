#### <a name="exactlyonce-and-durable-properties"></a><span data-ttu-id="8c6cd-153">ExactlyOnce 和 Durable 属性</span><span class="sxs-lookup"><span data-stu-id="8c6cd-153">ExactlyOnce and Durable Properties</span></span>
