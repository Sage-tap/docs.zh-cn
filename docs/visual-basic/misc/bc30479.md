# <a name="shared-sub-new-cannot-have-any-parameters"></a>共享的 "Sub New" 不能具有任何参数

在共享 `Sub New` 语句中包括了一个或多个参数，此方式无效。  
  
 **错误 ID：** BC30479  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除参数。
