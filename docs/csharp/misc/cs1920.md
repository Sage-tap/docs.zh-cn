# <a name="compiler-error-cs1920"></a><span data-ttu-id="fb305-103">编译器错误 CS1920</span><span class="sxs-lookup"><span data-stu-id="fb305-103">Compiler Error CS1920</span></span>

<span data-ttu-id="fb305-104">元素初始值设定项不能为空。</span><span class="sxs-lookup"><span data-stu-id="fb305-104">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="fb305-105">集合初始值设定项包含一个元素初始值设定项的序列。</span><span class="sxs-lookup"><span data-stu-id="fb305-105">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="fb305-106">元素初始值设定项不需要包含在大括号中，除非其包含赋值表达式。</span><span class="sxs-lookup"><span data-stu-id="fb305-106">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="fb305-107">但是，如果提供有大括号，则它们不能为空。</span><span class="sxs-lookup"><span data-stu-id="fb305-107">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="fb305-108">如果元素初始值设定项是对象初始值设定项，只要该初始值设定项包含新的对象创建表达式，大括号可能为空。</span><span class="sxs-lookup"><span data-stu-id="fb305-108">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fb305-109">更正此错误</span><span class="sxs-lookup"><span data-stu-id="fb305-109">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="fb305-112">示例</span><span class="sxs-lookup"><span data-stu-id="fb305-112">Example</span></span>  

 <span data-ttu-id="fb305-113">以下示例生成 CS1920：</span><span class="sxs-lookup"><span data-stu-id="fb305-113">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
