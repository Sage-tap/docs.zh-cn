# <a name="option-optionname-can-be-followed-only-by--or--"></a>选项 \<optionname> 后面只能跟“+”或“-”

该选项后的文本不是加号 (+) 或减号 (-)。  
  
 **错误 ID：** BC2009  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将此选项后的文本替换为加号 (+) 以打开该选项。  
  
     — 或 —  
  
- 将此选项后的文本替换为减号 (-) 以关闭该选项。
