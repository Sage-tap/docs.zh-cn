# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>“Using”语句在即时窗口中无效

`Using` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30186  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从“即时” `Using`**窗口中删除** 语句。
