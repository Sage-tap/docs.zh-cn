# <a name="methods-in-a-module-cannot-implement-interface-members"></a>模块中的方法无法实现接口成员

接口成员必须由类或结构实现。  
  
 **错误 ID：** BC31083  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从类或结构实现接口成员。
