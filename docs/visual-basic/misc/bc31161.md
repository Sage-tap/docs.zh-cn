# <a name="expected-closing----for-xml-comment"></a>XML 注释需要有结束的 "-->"

XML 注释文本缺少必需的结束字符“-->”。  
  
 **错误 ID：** BC31161  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 向 XML 注释文本末尾添加必需的结束字符“-->”。
