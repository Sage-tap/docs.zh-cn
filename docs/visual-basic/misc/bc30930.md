# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>\<name>无法找到编译此构造所需的 ""

 **错误 ID：** BC30930  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 提供必要的帮助程序元素。  
  
2. 确保代码是有效的托管代码。
