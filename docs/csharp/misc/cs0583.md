# <a name="compiler-error-cs0583"></a>编译器错误 CS0583

内部编译器错误。 编译器中发生了内部错误。 若要解决此问题，请尝试简化或更改以上所列位置附近的程序。 列表顶部的位置更接近发生内部错误的点。

> [!NOTE]
> Roslyn 中不再使用此编译器错误。
