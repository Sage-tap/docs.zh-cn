# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>“Case”语句在即时窗口中无效

`Case` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30719  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从“即时” `Case`**窗口中删除** 语句。
