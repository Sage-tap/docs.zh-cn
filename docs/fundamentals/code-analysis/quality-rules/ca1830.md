## <a name="how-to-fix-violations"></a><span data-ttu-id="8db6d-116">如何解决冲突</span><span class="sxs-lookup"><span data-stu-id="8db6d-116">How to fix violations</span></span>

<span data-ttu-id="8db6d-117">从调用中删除不必要的 `ToString()` 。</span><span class="sxs-lookup"><span data-stu-id="8db6d-117">Delete the unnecessary `ToString()` from the invocation.</span></span>

```csharp
using System.Text;

class C
{
    int _value;

    // Violation
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value.ToString()).AppendLine();
    }

    // Fixed
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value).AppendLine();
    }
}
```
